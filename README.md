I built a simple rule-based monitoring system for a compact smart device placed in a real indoor environment, where the goal was to detect short environmental or nearby-object changes and report them as event labels. In my Arduino program, I used humidity and temperature from the HS3003 sensor, magnetic disturbance from the magnetometer, and ambient/color response from the APDS9960 RGB and clear channels to detect exactly four events: BASELINE_NORMAL, BREATH_OR_WARM_AIR_EVENT, MAGNETIC_DISTURBANCE_EVENT, and LIGHT_OR_COLOR_CHANGE_EVENT.
